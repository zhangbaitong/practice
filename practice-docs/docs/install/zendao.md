##安装

1.下载：wget http://dl.zentao.net/5.3/ZenTaoPMS.5.3.stable.zip

2.解压：unzip ZenTaoPMS.5.3.stable.zip 

3.移动到指定目录(由于压缩后包含www，需要使用pms/www来使用，如何直接使用暂不考虑)

4.生成配置文件/www/htdocs/pm/config/my.php

5.配置管理员

6.删除install.php
