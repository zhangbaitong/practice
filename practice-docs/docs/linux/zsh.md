zsh

查看当前shell种类：

cat /etc/shells

/bin/bash
/bin/csh
/bin/ksh
/bin/sh
/bin/tcsh
/bin/zsh

切换到zsh

chsh -s /bin/zsh

配置相关信息

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
