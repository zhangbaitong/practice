1：环境说明
   zookeeper     10.122.75.229
   mesos-master  10.122.75.229
   mesos-slave  10.122.75.229,10.122.75.231
   marathon      10.122.75.229  

1：安装
	1)zookeeper
		sudo rpm -Uvh http://archive.cloudera.com/cdh4/one-click-install/redhat/6/x86_64/cloudera-cdh-4-0.x86_64.rpm
		sudo yum -y install zookeeper

	2)mesos marathon
		sudo rpm -Uvh http://repos.mesosphere.io/el/6/noarch/RPMS/mesosphere-el-repo-6-2.noarch.rpm
		sudo yum -y install mesos marathon


2：配置：
	1)zookeeper
		zookeeper-server-initialize --myid=1
  
		vi /etc/zookeeper/conf/zoo.cfg，添加
			server.1=10.122.75.229:2888:3888

	2）mesos	
		vi /etc/mesos/zk，添加
			zk://10.122.75.229:2181,10.122.75.231:2181/mesos
		vi /etc/mesos-master/quorum，添加
			2


3:启动
	1）zookeeper
		zookeeper-server start

	2）mesos-master 
		mesos-master --ip=10.122.75.229 --work_dir=/var/lib/mesos
		访问URL  http://117.78.19.76:5050/
		
	3）mesos-slave
		slave1
			mesos-slave --master=10.122.75.229:5050
		slave2
			mesos-slave --master=10.122.75.229:5050 --ip=10.122.75.231

	4）marathon	
		/root/code/marathon/bin/start --master zk://10.122.75.229:2181,10.122.75.231:2181/mesos --zk zk://10.122.75.229:2181,10.122.75.231:2181/marathon
		访问URL  http://117.78.19.76:8080/
		
	5）打开端口
		/sbin/iptables -I INPUT -p tcp --dport 5050 -j ACCEPT
		/sbin/iptables -I INPUT -p tcp --dport 5051 -j ACCEPT
		/sbin/iptables -I INPUT -p tcp --dport 8080 -j ACCEPT

4：启动docker容器（未成功）

		echo 'docker,mesos' > /etc/mesos-slave/containerizers

	2）重启mesos-slave（10.122.75.229,10.122.75.231）
