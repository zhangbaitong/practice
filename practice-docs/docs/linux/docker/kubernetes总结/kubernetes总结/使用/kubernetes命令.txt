1：创建pod、service、control(pod、service、control详见《kubernetes主要概念》，file.json参考《kubernetes json文件》)
    1）kubectl create -f <file.json>
    2）cat <file.json> | kubectl create -f -


2：查看pod、service、control
    1）kubectl get pod|service|control


3：删除pod、service、control
    1）kubectl delete -f <file.json>
    2）kubectl delete pod|service|control <id>
       注：<id>是2操作查询结果中的name字段


4：修改pod、service、control（可以通过增加资源的数量来达到集群扩容的目的）
    1）kubectl update -f <file.json>
    2）cat <file.json> | kubectl update -f -


5：查看pod、service、control的详细信息
    1）kubectl describe pod|service|control <id>


6：打印pod中某个container的日志信息
    1）kubectl log <pod>  <container>


7：设置kubernetes当前命名空间
    1）kubectl namespace <namespace>


8：查看当前命名空间
    1）kubectl namespace
