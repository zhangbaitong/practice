根据Dockerfile生成image
docker build -t="centos7-nginx-php-chen" .


在mysql容器中创建数据卷
docker  run -d  -p 80:80  -v /www/business_card/frontend:/www/business_card/frontend  --name nginx-php-chen  centos7-nginx-php-chen



检查一下修改过的配置文件是否正确
/usr/nginx/sbin/nginx -t

平滑重启nginx
/usr/nginx/sbin/nginx -s reload
