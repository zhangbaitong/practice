 19 ### 本地使用Git与github交互
 20 
 21 安装Git(Mac默认安装)
 22 
 23 设置Git用户和邮件：
 24     git config --global user.name "yourname"
 25     git config --global user.email yourname@xx.com
 26 
 27 本地使用代码库：
 28 
 29 获取仓库代码到本地：git clone https://github.com/zhangbaitong/practice.git
 30 
 31 更新仓库代码到本地：git pull origin master
 32 
 33 提交本地代码到仓库：
 34     git add .
 35     git commit -m "your message"
 36     git push origin master
 37 
 38 本地代码重命名提交：git add . -A
