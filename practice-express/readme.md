技术说明：基于HTML5的纯HTML移动端自适应的APP。

##使用说明

1.放到任何HTML容器中都可以正常运行。

2.如果想使用真实的数据，需要有PHP的支持

3.如果使用Mac，那么切换到WWW目录执行下面内容即可：

    python -m SimpleHTTPServer 8000
    访问：http://localhost:8000/

    测试数据：
    快递公司名称    单号
    天天快递        1
    申通快递        1

    同时可以从浏览器看到部分日志
